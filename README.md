## 目录

application 包括微服务服务程序

gateway 服务网关

modules 微服务模块

    base: 系统基础，数据字典，组织，区域
    public: 公用模块
            Unite 统一通用库

vue 后台 vue-element-admin typescript
layui 后台layui
    前端框架计划写一套阿里的[ant.design](https://ant.design/index-cn),这框架支持(React,Angular,Vue)，更适合企业级前端开发 (代码量大)
